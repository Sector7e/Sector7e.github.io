First Post After Hugo Update

August 11, 2022

The Sector 7e blog is now transitioning over to Hugo, and I will be working on transferring over the old content soon. The old blog relied on WordPress which is a common target for hackers and bots to compromise or deface for malicious reasons, so I decided this was a good opportunity to learn the static site generator framework known as Hugo. Because the blog will be generated as a static HTML site with many of the same features as WordPress, but it will be less likely to be compromised than a dynamic content management system that contains a dynamic web application that can execute code and a MySQL database. In addition, I’ll be switching over to GitHub as a host to keep this blog separate from my business hosting to reduce the attack surface of that site, but I will also utilize this as a learning opportunity for GitHub. I will be completing my bachelors degree in the next 9 months, and I will continue to document my personal projects like my custom built MAME arcades and other technology related projects and tutorials.
